Lista L1 - Exercício 01 - Envio de Dados ao Servidor com PHP

Exercício 01- Elabore um script em PHP que receba, de um formulário, duas notas e seus respectivos pesos, relativos à unidade curricular Programação WEB, e o nome do aluno. Em seguida, o script deve calcular a média ponderada do aluno usando a seguinte fórmula: Média ponderada = (nota * peso1 + nota2 * peso2) / (soma dos pesos). Faça com que o PHP gere, como resposta, a seguinte mensagem na página web:"Caro(a) aluno(a) $nomeDoAluno, de acordo com os dados fornecidos, sua média na unidade Programação Web é $media”.
