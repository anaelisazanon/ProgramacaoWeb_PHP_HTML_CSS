Lista L2 - Exercício 09 - Tratamento de vetores na linguagem PHP - exercicios de aprendizagem

Exercício 09 - Criar uma aplicação web destinada a auxiliar a compra de gêneros alimentícios on-line. Em um formulário, o cliente pode escolher até quatro produtos quaisquer, por meio de elementos checkbox. Em seguida, um script em PHP deverá criar e cadastrar, em um vetor de índice associativo, o preço de cada produto a ser adquirido. Use o nome do produto de cada checkbox como índice associativo deste vetor. Também, o formulário deve apresentar um checkbox para que o cliente informe se está pagando a compra feita com cartão de fidelidade do estabelecimento. Se sim, o cliente terá direito a um desconto de 5% em relação ao valor inicial da compra. Finalmente, o script em PHP deverá mostrar na página web o valor final da compra efetuada pelo cliente.
