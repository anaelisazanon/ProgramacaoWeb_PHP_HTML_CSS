Lista L2 - Exercício 02 - Tratamento de vetores na linguagem PHP - exercicios de aprendizagem

Exercício 02 - Elabore um script em PHP que receba, de um formulário, o nome e a nota de cada aluno na primeira avaliação de Programação Web. Cadastre os dados de três alunos. 0 script em PHP deverá:
- Armazenar estas notas em um vetor de índice associativo, usando o nome do aluno como índice;
- Mostrar, na página web, no formato tabular, os dados de cada aluno;
- Mostrar, na página web, o nome e a nota do aluno com a maior nota dentre os cadastrados.
